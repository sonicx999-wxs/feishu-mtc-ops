# 工具降级策略矩阵

本文档提供飞书 MTC 模式下工具降级的详细策略。

## 一、降级原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **身份优先** | 根据需要的身份选择工具 | User 身份优先 CLI，App 身份优先脚本/MCP |
| **可用性降级** | 主方案失败自动切换备选 | Python → Shell → MCP |
| **兼容性优先** | 优先选择兼容当前环境的方案 | 代理环境优先 Python |
| **最小依赖** | 优先选择依赖最少的方案 | 脚本 > MCP（需服务）> CLI（需授权） |
| **错误透明** | 每次降级记录原因 | 便于用户理解当前状况 |

## 三、降级触发条件

| 触发条件 | 检测信号 | 错误码 | 降级动作 |
|---------|---------|--------|---------|
| **CLI 授权失效** | `identity: "bot"`, 弹出验证 | — | 引导用户验证，或降级到脚本 |
| **CLI 工具失败** | `ok: false`, 错误信息 | 多种 | 切换到脚本或 MCP |
| **MCP 服务未启动** | `Connection closed` | — | 降级到脚本 |
| **MCP 工具缺失** | `Tool not available` | — | 降级到脚本 |
| **脚本网络超时** | `ETIMEDOUT` | — | 尝试其他脚本或 MCP |
| **权限不足** | 权限相关错误码 | 230027, 1063003 | 尝试其他身份或工具 |
| **资源不存在** | 资源相关错误码 | 99991663 | 提示用户确认资源 |

## 四、降级执行流程

```
执行操作 X
    │
    ├── 确认需要的身份（User / App）
    │
    ├── 确定可用工具集（根据环境预检）
    │
    ├── 选择主方案
    │
    ▼
执行主方案
    │
    ├── ✅ 成功 → 记录成功日志 → 完成
    │
    └── ❌ 失败
        │
        ├── 检查失败原因
        │
        ├── 判断是否可降级
        │
        ├── ✅ 可降级
        │   │
        │   ├── 记录降级原因
        │   │
        │   ├── 选择降级方案
        │   │
        │   ├── 执行降级方案
        │   │
        │   └── 循环直到成功或无可降级
        │
        └── ❌ 不可降级
            │
            ├── 记录完整错误信息
            │
            └── 返回用户提示
```

## 五、各环境类型的工具使用策略

### 5.1 类型A：完整环境（CLI ✅ / MCP ✅ / 脚本 ✅）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 文档类（User） | CLI → 脚本 → MCP | CLI 最简洁 |
| 聊天类（App） | Python脚本 → MCP → Shell | Python 最兼容 |
| 表格类 | CLI → MCP → 脚本 | CLI 最直接 |
| 权限类 | MCP → 脚本 | MCP 最标准 |

### 5.2 类型B：MCP失败（CLI ✅ / MCP ❌ / 脚本 ✅）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 文档类（User） | CLI → 脚本 | CLI 优先，MCP 不可用 |
| 聊天类（App） | Python脚本 → Shell | 使用脚本替代 MCP |
| 权限类 | 脚本 → CLI | 使用脚本替代 MCP |

### 5.3 类型C：CLI未授权（CLI ❌ / MCP ✅ / 脚本 ✅）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 文档类（User） | 脚本(API) → 提示授权 | 使用脚本的 App 身份创建 |
| 聊天类（App） | Python脚本 → MCP | 使用 App 身份操作 |
| 权限类 | MCP → 脚本 | MCP 可用 |

### 5.4 类型D：仅脚本可用（CLI ❌ / MCP ❌ / 脚本 ✅）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 文档类（App） | Python脚本 | 使用脚本的 App 身份创建 |
| 聊天类（App） | Python脚本 → Shell | 脚本是唯一方案 |
| 权限类 | Python脚本 | 使用脚本添加权限 |

### 5.5 类型E：仅CLI可用（CLI ✅ / MCP ❌ / 脚本 ❌）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 文档类（User） | CLI | CLI 是唯一方案 |
| 聊天类 | ❌ 不可用 | **CLI 禁止发送消息**，提示用户配置 MCP |
| 权限类 | CLI（有限制） | CLI 权限操作有限制 |

### 5.6 类型F：仅MCP可用（CLI ❌ / MCP ✅ / 脚本 ❌）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 文档类（App） | ❌ 不可用 | MCP 无法创建文档 |
| 聊天类 | MCP → 提示配置脚本 | MCP 是唯一可用方案 |
| 权限类 | MCP | MCP 可用 |

### 5.7 类型G：完全不可用（CLI ❌ / MCP ❌ / 脚本 ❌）

| 操作类型 | 推荐顺序 | 说明 |
|---------|---------|------|
| 所有操作 | ❌ 不可用 | 提示用户配置环境 |

## 七、错误码速查表

| 错误码 | 含义 | 建议降级动作 |
|--------|------|-------------|
| 230027 | Permission denied | 切换到 App 身份工具 |
| 232011 | Operator not in chat | 使用 App 身份操作群聊 |
| 1063003 | Invalid operation | 使用 App 身份创建文档 |
| 99991663 | 群聊不存在 | 确认群聊 ID 或创建群聊 |
| 99991661 | 无权限发送 | 确认应用是否在群聊中 |
| 99992361 | open_id cross app | 获取正确的 App 身份 open_id |
| 99991672 | Access denied | 开通所需权限 |
| ETIMEDOUT | 网络超时 | 检查代理配置，尝试其他工具 |
